# Stop Vibe Coding — A Guide to Agentic Centered Development

Presentation for **Coimbra.js 2026** by J. Pedro Oliveira.

Covers CLAUDE.md, Skills, Slash Commands, Subagents, Hooks, and Plugins — the building blocks for structuring projects so AI agents are productive from the first moment.

## Running locally

```bash
npm install
npm run dev
```

Open [localhost:3030](http://localhost:3030)
